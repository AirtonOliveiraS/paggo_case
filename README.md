# Case Paggo_OCR Front-End


##### Clone o repositório:
```sh
$ git clone https://github.com/AirtonOliveiraS/paggo_case.git

##### Acesse a pasta :
```sh
$ cd paggo_case
```
### Instalação
---

```bash
npm install
#&
npm build 
# or
npm run dev

```
Abrir 
[http://localhost:3000] (http://localhost:3000) para acompanhar o resultado.
